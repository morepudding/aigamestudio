## Convention de stockage

- `raw` : essais bruts generes, conserves pour comparaison ou reprise
- `approved` : version normalisee retenue pour le front et la composition

Convention de nommage actuelle :

- `editor-assets/<asset>-v<variant>-v4.png` pour le brut principal
- `editor-assets/<asset>-v<variant>-a<attempt>-v4.png` pour les autres essais bruts
- `editor-assets/<asset>-v<variant>-approved-v4.png` pour la version validee
